Optional source location override.
  • If type is set to BITBUCKET, then this value will be the HTTPS clone URL to the repository that contains the source and the build spec. Also, you must connect your AWS account to your Bitbucket account in CodeBuild Console.
  • If type is set to GITHUB, then this value will be the HTTPS clone URL to the repository that contains the source and the build spec. Also, you must connect your AWS account to your GitHub account in CodeBuild Console.
  • If type is set to GITHUB_ENTERPRISE, then this value will be the HTTPS clone URL to the repository that contains the source and the build spec. Also, you must save your personal access token in CodeBuild Console.
  • If type is set to CODECOMMIT, then this value will be the HTTPS clone URL to the repository that contains the source and the build spec(for example, https://git-codecommit.region-ID.amazonaws.com/v1/repos/repo-name ).
  • If type is set to S3, this is the path to the ZIP file that contains the source code(for example, bucket-name/path/to/object-name.zip).
