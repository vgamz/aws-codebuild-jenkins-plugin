Optional environment image override. The ID of the Docker image to use for this build.
  • To use a Docker image managed by AWS CodeBuild, the image ID should be the image identifier of Codebuild, for example: aws/codebuild/java:openjdk-8.
  • To use a Docker image in your Amazon ECR repository, the image ID format should be: account-ID.dkr.ecr.region.amazonaws.com/your-Amazon-ECR-repo-name:image-tag.
  • To use a Docker image in Docker Hub, the image ID format should be: repository-name/image-name:image-tag.
